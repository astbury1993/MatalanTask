== README

Hello and welcome to my Github page for this task. I have been working on creating a blog that allows a registered user to create articles that can then be edited, commented on and deleted. There is also some functionality in place to allow an external user to subscribe to a blogger by entering their email address.

I have hosted this application at https://curtisblog.herokuapp.com

I have used a basic Bootstrap front end.


<tt>rake doc:app</tt>.
